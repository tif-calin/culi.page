Hey, so this prototype is deprecated. Please check out the new and improved version of votevote at votevote.page. The goal for votevote is to be able to simulate a large number of electoral voting systems from a single election. This includes systems like FPTP, Borda Count, STAR Voting, Approval, Combined Approval, Condorcet, Copeland, River, Quadratic, and many other more niche ones.
